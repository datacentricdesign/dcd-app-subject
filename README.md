# dcd-data-subject

## production mode

- `npm run prod`
- go to `http://localhost:8080/subject/`

## development mode : watch

2 terminals :

- `npm run watch:browser` in one
- `npm run dev` in another
- go to `http://localhost:8080/subject`
- refresh page each change
